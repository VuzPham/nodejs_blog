Nodemon là một công cụ giúp tự động khởi động lại ứng dụng Node.js khi phát hiện có sự thay đổi trong mã nguồn. 
Điều này rất hữu ích trong quá trình phát triển, vì bạn không cần phải dừng và khởi động lại server thủ công 
mỗi khi chỉnh sửa code.

Express-Handlebars là một template engine phổ biến cho ứng dụng Node.js dùng với Express. 
Nó cho phép bạn tạo các trang HTML động với cú pháp tương tự như Mustache nhưng mạnh mẽ hơn.

Express là một framework web phổ biến cho Node.js, giúp xây dựng các ứng dụng web và API một cách nhanh chóng và dễ dàng.

Node-Sass là một thư viện giúp biên dịch file SCSS hoặc Sass thành CSS trong các dự án Node.js.

Mongoose là gì?
Mongoose là một thư viện ODM (Object Data Modeling) cho MongoDB và Node.js. Nó cung cấp một cách dễ dàng để làm việc 
với cơ sở dữ liệu MongoDB bằng cách sử dụng các Schema và Model.
npm install mongoose
